
# 📘 BMM350 Library Overview

 This driver provides a high-level interface to the Bosch BMM350 magnetometer.
---
# Overview
This library uses:

✅ **I2C as communication interface**.  
✅ **Bosch API as a reference in configuration and calculations**.  
✅ **YAW data**
✅ **data ready Interrupt**
✅ **Adaptive filtering**.  
✅ **Raw and compensated magnetic field reading** with detailed status codes.

---

| File         | Description                                 |
